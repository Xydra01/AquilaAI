# Slay the Spire 2 Latest Beta Patch Report (v0.103.0)

## 📅 Release Date
Latest beta patch notes available for **v0.103.0**

---

## 🔥 Major Changes & Fixes

### ⚖️ Leaderboard System Overhaul
- **Revamped leaderboard system** with two major changes to how scoring works for Daily Leaderboards
- Changed Acrobatics card: rarity increased from Common → Uncommon

### 🃏 Card Reworks
- **Follow Through**: Completely reworked
  - New card text: `Uncommon - Attack - Cost 1 | Deal 6(8) damage to ALL enemies.`

### 🔧 Bug Fixes
| Issue | Fix |
|-------|-----|
| Playing potions in multiplayer when dead | Fixed |
| Belt Buckle relic removing Dexterity incorrectly for other players who get potions | Fixed |
| Various balance adjustments across the game | Implemented |

### 🎨 Content Updates
- New art assets added throughout the game

---

## 📊 Previous Notable Patches (Reference)

### v0.101.0 Changes
- **Skulking Colony Nerf**: 
  - Deals less damage
  - Damage scales slower
  - No longer applies Dazed status cards
- **Haunted Ship**: Now applies different effects
